Chapter 1

Introducing SANLab-CM

This documentation is designed to introduce individuals to SANLab-CM's interface for the development of CPM-GOMS models and the execution of these models. This chapter will discuss the following topics:


  • The SANLab-CM Interface - An introduction to the SANLab-CM interface, its features, and some of the terminology used by SANLab-CM and its analogues in the literature
  • Instantiating Activities - Learn about activity types, creating activities in a SANLab-CM model, and how to adjust activity parameters.
  • Using Interactive Routines - Learn about interactive routines and how they can assist in the creation and management of a model.
  • Building a Basic Model - A step-by-step walkthrough building a simple model via task analysis using all of the tools discussed so far.
  • Running a Model - A discussion about running a model and some of the tools available to control model execution.
  • The Results Window - Learn about the histogram and profile tool to look at the model output.
  • Analyzing Critical Paths - Learn about the tools SANLab-CM offers to compare critical paths for a model.
  • Activity Criticality - A look at tools to examine how often activities and interactive routines appear on the model's critical path.
