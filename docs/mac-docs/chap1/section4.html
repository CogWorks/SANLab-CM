Chapter 1

Building a Basic Model

This section will show how activities and interactive routines can be used to build a CPM-GOMS model from the ground up. The model itself will be based on the first telephone operator & assistance (TAO) call type modeled by Gray, John, & Atwood (1993). Below is a plain-text description of the task:

The TAO would wait for the terminal to signal an incoming call. Upon hearing the arrival tone, the TAO would look for a pair of symbols which encoded whether the call was initiated from a private telephone or a public telephone. Given this information, the TAO would greet the caller either with the statement "New England Telephone, may I help you?" or "New England Public Telephone, may I help you?". In this particular case, the customer would request that the call be made collect. In response to this, the operator would need to press the STA COL button, write the customer's name on a notepad, and verify the billing information on the screen. The system would ring until the individual being called answered. The TAO would query the callee if her or she would accept the charges for the collect call. Upon receiving an affirmative answer, the TAO presses the POS-REL to release the workstation and allow for a new call.

First, it is necessary to build a standard GOMS representation of this task. Since this task is relatively serial in its execution, it can be assumed that there is a single goal and a single method with multiple submethods. For operators, there are both visual and auditory perception, speech, left-hand and right-hand key presses, writing (using the right hand), and eye movements. Here is a rough first method:

GOAL: HANDLE-CALL

METHOD for HANDLE-CALL:

  1. Listen for CAT
  2. Look for symbols
  3. Identify call type
  4. Greet customer (GREET)
  5. Listen for customer saying "I would like to place a collect call from John Smith to 518-555-5555"
  6. Press the STA COL button to identify collect call
  7. Write customer's name down
  8. Enter telephone number
  9. Verify billing information
  10. Wait for individual to answer call
  11. Prompt caller (PROMPT)
  12. Press the POS-REL button to release workstation
METHOD for GREET:

  1. Say "New England Public Telephone, may I help you?"
