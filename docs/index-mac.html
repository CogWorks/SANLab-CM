SANLab-CM for Mac OS X



SANLab-CM is a tool for cognitive modelers to rapidly create CPM-GOMS models in order to predict how humans interact with external systems as well as construct models of existing human data as a measure of validity checking. This manual is designed to introduce SANLab-CM to beginners and act as a reference for the many features which have been incorporated into SANLab-CM in order to assist in modeling.
